This codebase allows a 3ds to control Rokenbok vehicles through an arduino.
It requires the old Rokenbok toys with the wired controllers and control tower,
not the newer ones with IR controllers. 3dsCode has the advanced version with
live video streaming, but only runs on New 3DS or New 2DS models, not the original
ones. The code in Old3dsCode works on older models but does not feature video streaming.
